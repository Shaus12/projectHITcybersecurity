## Project Description
A machine-learning based system for detecting malicious executables using structural metadata from PE (Portable Executable) headers.

## AI Model
- **Algorithm:** Random Forest Classifier (100 estimators)
- **Features:** VirtualSize, Entropy, NumberOfSections, Characteristics

## Security Testing
Adversarial robustness testing performed using incremental feature manipulation.

## MVP Features
- Real-time classification UI (Streamlit)
- Persistent logging for auditing (`traffic_logs.log`)
- Visual detection alerts
